## Versions

- RabbitMQ: 3.7.12
- Delayed Message Plugin: 3.7.x
- Alpine Linux: 3.8

## Usage

Clone:

- ``` git clone https://github.com/danvincenzi/dockerfile.git ```
- ``` cd dockerfile/rabbitmq/alpine/3.7.12 ```

Build:

- ``` docker build --tag rabbitmq:3.7.12 . ```
